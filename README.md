# DCCWIFI Landing Page

Landing page premium, responsif, dan ramah SEO untuk **DCCWIFI** (PT Restu Panca Alam), provider internet cepat, stabil, dan unlimited di Kabupaten Ponorogo, Jawa Timur.

## 🚀 Fitur Utama
- **Aesthetic Tech-Modern:** Menggunakan grid visual, gradasi warna dinamis, interaksi halus, serta mode responsif.
- **FOUC Prevention:** Transisi visual halaman mulus (halaman hanya ditampilkan penuh saat CSS dan JS selesai dimuat).
- **Security First:** Dikonfigurasi dengan kebijakan keamanan ketat melalui Content Security Policy (CSP) dan Referrer Policy eksternal tanpa inline script.
- **SEO & Schema Markup:** Struktur semantic HTML5 lengkap dengan JSON-LD LocalBusiness Schema untuk peringkat pencarian lokal Ponorogo yang lebih baik.

## 🛠️ Tech Stack
- **Build Tool:** [Vite v6](https://vite.dev/)
- **CSS Framework:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons:** [Lucide Icons](https://lucide.dev/)
- **Fonts:** Google Fonts (Outfit & Inter)
